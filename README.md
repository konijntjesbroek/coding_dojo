## Coding Dojo
A repository of documentation and tools used for the Coding Dojo discord server

### Belt System
The duties and privilegs given to the various roles on the server.

### sensei\_bot
*In Development:*
- challenge - interface for training exercises
- calendar - interface for upcoming events
- news - give latest updates for the server
- projects - listing of active projects 
- showcase - demonstrations from community members
- eval - allow users to evaluate code, Possibly based off online interpreters
